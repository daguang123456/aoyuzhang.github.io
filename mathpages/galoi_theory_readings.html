\(\frac{1}{1}\) I never studied Galoi theory but I heard that it was used to prove that it is impossible to trisect all angles using only unmarked ruler and compass. Therefore it is very interesting to see why this is the case. We can relate geometry and algebraic numbers as follow:
A number is called constructible if if can be found using only an unmarked ruler and a compass.
We start by defining a field:
Given a set \(F\) with two binary operations \(+,*\), \(F\) is called a field if \((F,+)\) is a group with identity element \(0\) and if \((F-\{0\}, *)\) is a group with identity element \(1\) and the * operation is distributive over +.
For example, the set of numbers of the form \(Q[\sqrt{2}]= \{a+b\sqrt{2}\}\), where \(a,b\) are rational numbers, form a field under normal addition and multiplication operation for real numbers.
A number \(\alpha\) is called an algebraic number if it is the root of some polynomial \(p(x)\).
If \(a\) is an algebraic number then \(Q[a]\) is a field
A field extension of a field \(F\) is a field \(K\) containing \(F\)
